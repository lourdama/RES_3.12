## 📖 Description

Portfolio personnel présentant mes compétences, projets académiques et expériences professionnelles. Site entièrement fait main avec un design moderne « néon doux » et des animations personnalisées.

## ✨ Fonctionnalités

- 🌓 Mode Clair / Sombre avec transitions fluides
- 🌍 Support Multilingue (FR / EN)
- 🎨 Fond animé style réseau neuronal
- ⌨️ Titres avec animation de frappe
- 📱 Design responsive pour tous les appareils
- 🖼️ Galeries de projet avec système de lightbox
- 📄 Téléchargement de CV (3 versions)
- 📧 Formulaire de contact avec PHPMailer

## 🚀 Installation & Déploiement

### Prérequis
- Serveur web (Apache/Nginx) avec PHP 7.4+
- Extension PHP `openssl` pour l'envoi d'emails

### Installation locale
1. Clonez le repository :
   ```bash
   git clone https://github.com/lourdama/RES_3.12.git
   ```
2. Servez le dossier via un serveur local (ex: XAMPP, Live Server)
3. Ouvrez `index.html` dans un navigateur

### Configuration Email
Modifiez `php/sendmail.php` avec vos identifiants SMTP pour activer le formulaire de contact.
